- **[migrate-to-shoehorn](./migrate-to-shoehorn/SKILL.md)** — 将测试文件从 `as` 类型断言迁移到 @total-typescript/shoehorn。
- **[scaffold-exercises](./scaffold-exercises/SKILL.md)** — 创建练习目录结构，包括章节、问题、解决方案和说明。
- **[setup-pre-commit](./setup-pre-commit/SKILL.md)** — 使用 lint-staged、Prettier、类型检查和测试设置 Husky pre-commit 钩子。
