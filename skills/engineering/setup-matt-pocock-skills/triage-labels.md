# 分类标签

技能使用五个规范分类角色来沟通。此文件将这些角色映射到此仓库问题跟踪器中使用的实际标签字符串。

| mattpocock/skills 中的标签 | 我们跟踪器中的标签   | 含义                         |
| -------------------------- | -------------------- | ---------------------------- |
| `needs-triage`             | `needs-triage`       | 维护者需要评估此 Issue        |
| `needs-info`               | `needs-info`         | 等待报告者提供更多信息        |
| `ready-for-agent`          | `ready-for-agent`    | 完全指定，可供 AFK 智能体使用 |
| `ready-for-human`          | `ready-for-human`    | 需要人类实现                  |
| `wontfix`                  | `wontfix`            | 不会被处理                    |

当技能提到一个角色时（例如"应用 AFK-ready 分类标签"），使用此表中的对应标签字符串。

编辑右侧列以匹配你实际使用的任何词汇。
